# 🎤 Assistente de Voz com IA

Projeto desenvolvido em Python que permite conversar com uma inteligência artificial por voz.

## 🚀 Tecnologias utilizadas
- Python
- OpenAI (Speech-to-Text e ChatGPT)
- gTTS (Text-to-Speech)

## 🎯 Funcionalidades
- Captura de áudio do usuário
- Transcrição de voz para texto
- Geração de resposta com IA
- Conversão da resposta em áudio

## ▶️ Como executar

1. Instale as dependências:
pip install -r requirements.txt

2. Configure sua API KEY da OpenAI no código

3. Execute o projeto:
python app.py

## 💡 Observação
É necessário ter microfone ativo no computador.
